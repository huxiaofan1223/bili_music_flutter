# bili_music_flutter

A Audio Player source by Bilibili.

## 功能点
- 扫码登录
- 注销登录
- 搜索音频
- 缓存音频
- 播放本地音乐

### 更多

- 不用担心是否有音乐会员(b站上几乎什么歌都能听),安卓车机播放音乐方便
- 基于flutter，理论上可以跨多个平台(windows,linuc,mac,ios)，但我只做了安卓的适配
- 由于请求的API需要登录凭证，本软件强制要求扫码登录bilibili，但无其他用途
